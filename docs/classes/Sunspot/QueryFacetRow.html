Class Sunspot::QueryFacetRow

  1. lib/sunspot/query_facet_row.rb (view online)

Objects of this class encapsulate a single query facet row returned for a query facet.

Attributes

count [R] Number of documents in the result set that match this facet’s scope.
value [R] This is the “label” passed into the query facet row when it is defined in the search.

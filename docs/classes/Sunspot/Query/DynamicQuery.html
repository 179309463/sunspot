Class Sunspot::Query::DynamicQuery

  1. lib/sunspot/query/dynamic_query.rb (view online)
Parent: Sunspot::Query::FieldQuery

A dynamic query is a proxy object that implements the API of the FieldQuery class, but wraps a dynamic field factory and thus applies the query components using dynamic field instances. DynamicQuery instances are publicly generated by the Query#dynamic_query factory method.
